# Guía Empresarial

Una plataforma web para descubrir y conectar con negocios locales en Perú.

## Descripción

Guía Empresarial es un sitio web que permite a los usuarios buscar y explorar empresas locales, facilitando la conexión entre negocios y clientes.

## Características

- Búsqueda de empresas
- Listado de categorías comerciales
- Información de contacto
- Interfaz responsive

## Estructura del Proyecto

- **index.php** - Página principal
- **empresas.php** - Listado y búsqueda de empresas
- **categorias.php** - Categorías de negocios
- **contacto.php** - Formulario de contacto
- **about.php** - Información del sitio
- **assets/** - Recursos CSS, JavaScript e imágenes
- **includes/** - Componentes compartidos (header, footer)
- **db.php** - Configuración de base de datos

## Requisitos

- PHP 7.0+
- Servidor web (Apache/Nginx)
- Base de datos MySQL

## Instalación

1. Clonar repositorio
2. Configurar base de datos en `db.php`
3. Acceder al sitio en tu servidor local
